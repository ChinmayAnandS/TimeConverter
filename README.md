# TimeConverter

A simple and intuitive web application for converting time between different time zones. Easily convert local time to any global time zone with real-time updates.

## Features

- **Real-time Conversion**: Instantly convert time between multiple time zones
- **User-friendly Interface**: Clean, responsive design that works on desktop and mobile
- **Timezone Selection**: Choose from a comprehensive list of world time zones
- **Accurate Calculations**: Handles daylight saving time and timezone offsets correctly

## Demo

View the live demo: [TimeConverter on GitHub Pages](https://ChinmayAnandS.github.io/TimeConverter/)

## Usage

1. Open `index.html` in your web browser
2. Select your local time zone
3. Enter the time you want to convert
4. Choose the target time zone
5. View the converted time instantly

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Project Structure

- `index.html` - Main HTML file
- `styles.css` - Styling for the application
- `script.js` - Main JavaScript logic
- `timezones.js` - Timezone data and utilities
